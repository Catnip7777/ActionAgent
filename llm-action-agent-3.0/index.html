动作格式

        

          

            大模型在回复中嵌入「动作块」（内含 JSON），本页配置如何从大段文字里识别并提取这些动作块。
            默认使用 <action>{JSON}</action>；若你的模型更习惯 Markdown 代码块或自定义标签，在此切换并测试解析。
            格式说明也会写入系统提示词，告诉模型应以何种形式输出动作。
            在动作块开始标签前加转义前缀（默认反斜杠 \）可让该块仅作展示、不被执行。
            解析器从最外层闭合标签匹配 JSON，因此 JSON 字符串字段内可含 </action> 等字面量（如正则、代码），无需额外转义。
          

          

            
            
          

          

            
            
          

          

            
            
          

          

            
            
            动作块开始标签前加此前缀则不执行，如 \…
