# Action Agent Lite 分支介绍

> 纯浏览器端的 AI 助手交互平台

## 核心特性

### 1. 智能对话
通过聊天界面与大模型（OpenAI 兼容接口）交互，支持流式输出、思考过程展示、Token 用量统计。

### 2. 可执行动作
AI 可以输出 JSON 动作块，由浏览器本地执行，支持：

- **文件操作**：读写、追加、删除、编辑、建目录
- **浏览器操作**：打开链接、剪贴板、下载、localStorage、通知
- **任务管理**：增删改查、分解子任务

### 3. 工作区管理
支持多工作区，通过浏览器文件系统 API 授权文件夹，可设置读写/只读权限。

### 4. 任务板
每个对话可独立管理任务列表，AI 可自动分解和追踪进度。

### 5. 对话管理
多对话切换、分组、自动命名、导出聊天记录。

### 6. API 设置
支持 OpenAI 兼容接口，可配置模型、温度、最大 Token 等。

### 7. 本地存储
配置和对话历史优先使用 IndexedDB，降级到 localStorage。
